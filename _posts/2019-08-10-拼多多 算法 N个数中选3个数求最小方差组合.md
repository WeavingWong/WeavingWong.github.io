```python
-------------------第一题---------------
给定N个整数，找出3个数，满足这三个数的组合时所有组合中方差最小的。
输入: 第一行整数N，第二行N个整数，其中所有数绝对值小于100000.
```


```python

def get_result(n,numbers):
    numbers = sorted(numbers)
    average=(numbers[0]+numbers[1]+numbers[2])/3
    min_variance = ((numbers[0]-average)**2+(numbers[1]-average)**2+(numbers[2]-average)**2)/3
    for i in range(3,len(numbers),1):
        average=(numbers[i]+numbers[i-1]+numbers[i-2])/3
        min_variance_tmp = ((numbers[i]-average)**2+(numbers[i-1]-average)**2+(numbers[i-2]-average)**2)/3
        if min_variance_tmp < min_variance:
            min_variance=min_variance_tmp
    return min_variance
if __name__=='__main__':
    n = int(input())
    numbers = list(map(int,input().split()))
    res=get_result(n,numbers)
    print("%.2f" % res) # 这是考点，不能用round四舍五入
```
